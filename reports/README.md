# Pipeline Reports

This directory contains generated reports from CI/CD pipelines.

## Contents

- `coverage/` - HTML coverage reports (pytest-cov)
- `coverage.xml` - XML coverage for Codecov
- `lint-ruff.txt` - Ruff linting results
- `lint-mypy.txt` - MyPy type checking results
- `security-bandit.txt` - Bandit security scan results
- `quality-summary.md` - Aggregated quality gate summary

## Access

- Local: Open `coverage/index.html` in browser
- CI/CD: Download from GitHub Actions artifacts
