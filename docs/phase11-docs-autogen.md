# Phase 11.5: Documentation Auto-Generation

## Overview
Automated documentation generation for model cards, API docs, changelogs, and README.

## Components

1. Model Card Generator - Generates standardized model cards
2. API Doc Generator - Exports OpenAPI schema
3. Changelog Generator - Creates CHANGELOG.md
4. README Updater - Auto-updates project README

## Usage

Run the documentation generator:

    python scripts/phase11/p11.5-generate-all-docs.py

## Output

- docs/model_cards/ - Model cards (JSON + Markdown)
- docs/api/ - API documentation
- CHANGELOG.md - Project changelog
- README.md - Updated README
